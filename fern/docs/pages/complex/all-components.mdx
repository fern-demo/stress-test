### Basic Code Block

```python
def hello_world():
    """A simple hello world function."""
    print("Hello, World!")
    return True
```

### Code Block with Title

```javascript title="example.js"
function calculateSum(a, b) {
    return a + b;
}

console.log(calculateSum(5, 3));
```

### Code Block with Line Numbers

```typescript
interface User {
    id: number;
    name: string;
    email: string;
}

function getUser(id: number): User {
    return {
        id: id,
        name: "John Doe",
        email: "john@example.com"
    };
}
```

## Accordions

<Accordion title="What is Fern?">
Fern is a toolkit for building REST APIs. It includes tools for generating SDKs, documentation, and more.
</Accordion>

<Accordion title="How do I get started?">
Getting started with Fern is easy:

1. Install the Fern CLI
2. Initialize your project
3. Define your API
4. Generate your docs

```bash
npm install -g fern-api
fern init
fern generate --docs
```
</Accordion>

<Accordion title="What languages are supported?">
Fern supports multiple programming languages including:
- Python
- TypeScript
- JavaScript
- Java
- Go
- Ruby
- And more!
</Accordion>

## Steps

<img src="/docs/assets/favicon.png" alt="Favicon" />

<Steps>
  <Step title="Create an account">
    Sign up for a Fern account at [buildwithfern.com](https://buildwithfern.com).
  </Step>
  
  <Step title="Install the CLI">
    Install the Fern CLI globally:
    
    ```bash
    npm install -g fern-api
    ```
  </Step>
  
  <Step title="Initialize your project">
    Create a new Fern project:
    
    ```bash
    fern init
    ```
  </Step>
  
  <Step title="Define your API">
    Edit the API definition files in the `fern/` directory.
  </Step>
  
  <Step title="Generate documentation">
    Generate your documentation:
    
    ```bash
    fern generate --docs
    ```
  </Step>
</Steps>
